## 1. ゲーム概要

**ジャンル**: 冒険 × 放置ショップ × すごろくダンジョン  
**プレイスタイル**: 放置型ループゲーム。ダンジョンで装備を集め、ショップで売り、強くなってより深いダンジョンへ。

### コアループ

```
潜入 → 探索・戦闘 → 帰還 → 鑑定・強化・出品 → 放置販売 → 資金獲得 → 次の潜入
```

### 2-4. 罠マス（2d6ダイス判定）

```
ダイス2個（各1〜6）を振る
lukBonus = min(2, floor(luk / 10))
total = min(12, die1 + die2 + lukBonus)
```

| 結果 | 条件 | 効果 |
|---|---|---|
| 大成功 ✨ | 補正前 die1+die2 = 12 | ダメージなし |
| 成功 ✓ | total ≥ 成功ライン | ダメージなし |
| 失敗 ✕ | total < 成功ライン | maxHP×30〜45%ダメージ |
| 大失敗 💀 | die1=1かつdie2=1 | maxHP×40%ダメージ＋毒 |

成功ライン：1階=7 / 2階=8 / 3階=8 / 4階=9  
lukBonus上限：3〜4階=+2、4階のみ+3（毒スタートの補填）

### 2-5. 休息マス（2d6ダイス判定）

```
total = min(12, die1 + die2 + lukBonus)
```

| 合計 | 回復率 | 演出 |
|---|---|---|
| 2〜4 | 15% | 😓 落ち着けなかった… |
| 5〜7 | 20% | 😴 少し休んだ |
| 8〜10 | 30% | 💤 ぐっすり眠れた |
| 11 | 40% | 💚 よく休めた！ |
| 12 | 50% | ✨ 大当たり！ |

HP満タンの場合はwindfall（Gold）に自動変換。

## 3. 戦闘

### 3-1. プレイヤーの行動

- **攻撃**: 通常攻撃（クリティカル判定あり）
- **防御**: 次の敵攻撃ダメージ軽減
- **逃げる**: 成功率は素早さ依存

### 3-2. ダメージ計算

**プレイヤー→敵**
```
dmg = max(1, floor(str × 2 × variance - def × 0.7))  variance=0.85〜1.15
```

**敵→プレイヤー**
```
dmg = max(1, floor(str² × 5 / (def + str × 5)))
```

### 3-3. ボス戦

- フェーズ制（HP75%・50%・25%で変化）
- 怒りゲージ：被ダメ蓄積でスキル強化
- 登場モーダル・必殺技カットイン・撃破リザルトあり

### 3-3b. 戦闘スキル（MP消費）

スキルツリーで特定スキルを習得すると解禁。戦闘中に「✨ スキル」ボタンから発動。

| スキル | 解禁条件 | MP消費 | 効果 |
|---|---|---|---|
| 魔法弾 ✨ | 会心強化 | 6 | INT×3ダメージ（DEF無視） |
| 鬨の声 📯 | 血の昂り | 8 | 次の攻撃ATK×2.0 |
| 聖なる盾 🔆 | 要塞 | 6 | そのターンの被ダメを完全無効化 |
| 毒霧撒き ☠ | 毒刃 | 4 | 80%で敵に毒付与（毎ターンmaxHP×5%） |

習得スキルが1種の場合は直接ボタン表示、2種以上の場合はモーダル選択。

### 3-4. 状態異常

- **毒**: 移動・戦闘後に最大HP×3%ダメージ（poisonMul補正あり）。休息マスで解除。
- **熱傷**: 移動・戦闘後に最大HP×2%ダメージ。毒と別スタック。帰還で解除。業炎の覇道セット2部位で30%軽減。同セット3部位で攻撃時15%の確率で敵にも熱傷付与（敵熱傷: maxHp×3%/ターンのスリップ）。

SP消費でスキルを習得（前提スキルあり）。タブ：火力/防御/探索/永続。

**火力系主要スキル**
- 剛力: str+3
- 会心強化: クリ率+5%
- 処刑人: 敵HP30%以下で与ダメ+30%

**防御系主要スキル**
- 鉄の体: vit+3
- 不屈: 最大HP+10%

- 鷹の目: レア率+3%
- 財宝感知: 宝箱出現率+10%
- 逃走術: 逃走成功率+25%

### 4-4. 転生システム

転生条件: `ng >= 1`（5階または10階ボス撃破後フェーズ移行済み）かつ **Lv50以上**。転生後：
- レベル・ステータス・スキルをリセット
- 永続SP獲得（転生回数に応じて増加）
- 転生回数（worldRank）蓄積 → 敵が強化される

**引き継ぐもの（S46更新）**
- Gold / ショップ / 装備 / 在庫
- 永続スキル・永続SP
- 鞄拡張段階（lootSlots）← S46から引き継ぎに変更
- 真エンドフロア解禁状態（abyssFloorCleared）

## 6. ショップシステム

### 6-1. 基本

- 棚に陳列したアイテムが時間経過で自動販売
- ショップLv（EXP蓄積）でスタッフ雇用・棚枠拡張
- 流行システム：特定種類の売値・速度が30分間上昇

### 6-2. 販売速度（基準）

Normal=1分 / Rare=3分 / Epic=8分 / Legendary=20分

### 6-3. 裏取引（ブラックマーケット）

ショップLv5で解禁。呪い品・未鑑定品を高値売却。バイヤー4種が訪問。摘発リスクあり。

### 6-4. 常連客

10回購入で常連認定。Lvが上がると売値ボーナス・注文機能が解放。

**注文機能（Lv3解放・S109〜S117 完成）:**

| 機能 | 内容 | セッション |
|---|---|---|
| 注文バッジタップ→出品モーダル | タップで常連モーダルを閉じ、対象タイプフィルタ済みの一括出品モーダルを開く | S109 |
| 出品時フィードバックログ | 注文品を棚に出すとショップログ「⭐ [客名]の注文品を棚に出した！」を表示 | S110 |
| 出品済みバッジ切替 | 注文タイプが棚に出品済みなら「✅ 出品中」（緑）に切替 | S111 |
| 在庫なしヒント | フィルタ対象タイプの在庫がなければ「⚠ 在庫がありません」を表示 | S111 |
| タイムアウト即時更新 | 10分タイムアウト時、モーダル開放中なら注文バッジを即時更新 | S113 |
| 「まだかな」ログ抑制 | 注文品未在庫時のログを5分に1回まで制限 | S114 |
| 残り時間表示 | 未出品バッジに「⏱ あとN分」。残り2分以内は赤色 | S115 |
| 残り時間自動更新 | モーダル開放中30秒ごとに自動更新 | S116 |

**手動テスト手順:**
1. 常連Lv3以上で注文が来た状態 → バッジタップ → 対象タイプフィルタ済みモーダルが開く
2. 注文タイプのアイテムを棚に出す → バッジが緑「✅ 出品中」に切替わる
3. 注文タイプの在庫がない状態でバッジタップ → 「⚠ 在庫がありません」が表示される
4. 注文バッジに「⏱ あとN分」が表示される。30秒待って数値が更新される
5. 10分経過後「しばらく待ったけど…」ログが出てバッジが消える
6. **S119追加**: 注文タイプが `sub`/`cloak`/`seal` の場合もフィルタボタンが表示され、バッジタップで正しくフィルタ済みモーダルが開く（S119修正確認）

**バージョン**: S48

## 1. ゲーム概要

**ジャンル**: 冒険 × 放置ショップ × すごろくダンジョン  
**プレイスタイル**: 放置型ループゲーム。ダンジョンで装備を集め、ショップで売り、強くなってより深いダンジョンへ。

### コアループ

```
潜入 → 探索・戦闘 → 帰還 → 鑑定・強化・出品 → 放置販売 → 資金獲得 → 次の潜入
```

### 2-4. 罠マス（2d6ダイス判定）

```
ダイス2個（各1〜6）を振る
lukBonus = min(2, floor(luk / 10))
total = min(12, die1 + die2 + lukBonus)
```

| 結果 | 条件 | 効果 |
|---|---|---|
| 大成功 ✨ | 補正前 die1+die2 = 12 | ダメージなし |
| 成功 ✓ | total ≥ 成功ライン | ダメージなし |
| 失敗 ✕ | total < 成功ライン | maxHP×30〜45%ダメージ |
| 大失敗 💀 | die1=1かつdie2=1 | maxHP×40%ダメージ＋毒 |

成功ライン：1階=7 / 2階=8 / 3階=8 / 4階=9  
lukBonus上限：3〜4階=+2、4階のみ+3（毒スタートの補填）

### 2-5. 休息マス（2d6ダイス判定）

```
total = min(12, die1 + die2 + lukBonus)
```

| 合計 | 回復率 | 演出 |
|---|---|---|
| 2〜4 | 15% | 😓 落ち着けなかった… |
| 5〜7 | 20% | 😴 少し休んだ |
| 8〜10 | 30% | 💤 ぐっすり眠れた |
| 11 | 40% | 💚 よく休めた！ |
| 12 | 50% | ✨ 大当たり！ |

HP満タンの場合はwindfall（Gold）に自動変換。

## 3. 戦闘

### 3-1. プレイヤーの行動

- **攻撃**: 通常攻撃（クリティカル判定あり）
- **防御**: 次の敵攻撃ダメージ軽減
- **逃げる**: 成功率は素早さ依存

### 3-2. ダメージ計算

**プレイヤー→敵**
```
dmg = max(1, floor(str × 2 × variance - def × 0.7))  variance=0.85〜1.15
```

**敵→プレイヤー**
```
dmg = max(1, floor(str² × 5 / (def + str × 5)))
```

### 3-3. ボス戦

- フェーズ制（HP75%・50%・25%で変化）
- 怒りゲージ：被ダメ蓄積でスキル強化
- 登場モーダル・必殺技カットイン・撃破リザルトあり

### 3-3b. 戦闘スキル（MP消費）

スキルツリーで特定スキルを習得すると解禁。戦闘中に「✨ スキル」ボタンから発動。

| スキル | 解禁条件 | MP消費 | 効果 |
|---|---|---|---|
| 魔法弾 ✨ | 会心強化 | 6 | INT×3ダメージ（DEF無視） |
| 鬨の声 📯 | 血の昂り | 8 | 次の攻撃ATK×2.0 |
| 聖なる盾 🔆 | 要塞 | 6 | そのターンの被ダメを完全無効化 |
| 毒霧撒き ☠ | 毒刃 | 4 | 80%で敵に毒付与（毎ターンmaxHP×5%） |

習得スキルが1種の場合は直接ボタン表示、2種以上の場合はモーダル選択。

### 3-4. 状態異常

- **毒**: 移動・戦闘後に最大HP×3%ダメージ（poisonMul補正あり）。休息マスで解除。
- **熱傷**: 移動・戦闘後に最大HP×2%ダメージ。毒と別スタック。帰還で解除。業炎の覇道セット2部位で30%軽減。同セット3部位で攻撃時15%の確率で敵にも熱傷付与（敵熱傷: maxHp×3%/ターンのスリップ）。

## 4. プレイヤー成長

### 4-1. ステータス

| stat | 名称 | 主な効果 |
|---|---|---|
| str | 力 | 攻撃力 |
| def | 守 | 被ダメ軽減 |
| agi | 速 | 回避率・逃走率 |
| vit | 体 | 最大HP |
| int | 賢 | MP・魔法系 |
| luk | 運 | レア率・罠判定・クリティカル |

### 4-2. レベルアップ

- 敵を倒すとEXP獲得
- レベルアップ時にSP+1獲得（任意のステータスに振り分け）

### 4-3. スキルツリー

SP消費でスキルを習得（前提スキルあり）。タブ：火力/防御/探索/永続。

**火力系主要スキル**
- 剛力: str+3
- 会心強化: クリ率+5%
- 処刑人: 敵HP30%以下で与ダメ+30%
- 毒刃: 通常攻撃15%で毒付与

**防御系主要スキル**
- 鉄の体: vit+3
- 不屈: 最大HP+10%
- 反撃: 被ダメ5%で即反撃

**探索系主要スキル**
- 鷹の目: レア率+3%
- 財宝感知: 宝箱出現率+10%
- 逃走術: 逃走成功率+25%

### 4-4. 転生システム

転生条件: `ng >= 1`（5階または10階ボス撃破後フェーズ移行済み）かつ **Lv50以上**。転生後：
- レベル・ステータス・スキルをリセット
- 永続SP獲得（転生回数に応じて増加）
- 転生回数（worldRank）蓄積 → 敵が強化される

**引き継ぐもの（S46更新）**
- Gold / ショップ / 装備 / 在庫
- 永続スキル・永続SP
- 鞄拡張段階（lootSlots）← S46から引き継ぎに変更
- 真エンドフロア解禁状態（abyssFloorCleared）

**worldRank節目**

| 10 / 50 | ダンジョンログ1行 | なし |

## 6. ショップシステム

### 6-1. 基本

- 棚に陳列したアイテムが時間経過で自動販売
- ショップLv（EXP蓄積）でスタッフ雇用・棚枠拡張
- 流行システム：特定種類の売値・速度が30分間上昇

### 6-2. 販売速度（基準）

Normal=1分 / Rare=3分 / Epic=8分 / Legendary=20分

### 6-3. 裏取引（ブラックマーケット）

ショップLv5で解禁。呪い品・未鑑定品を高値売却。バイヤー4種が訪問。摘発リスクあり。

### 6-4. 常連客

10回購入で常連認定。Lvが上がると売値ボーナス・注文機能が解放。

### 16-2. エネミー固有スキル（specialSkill）仕様

```javascript
specialSkill: {
  id, name, msg,
  type,       // 効果タイプ（9種）
  rate,       // 通常発動率（%）
  rateEnraged // HP50%以下時の発動率（%）
  // typeごとの追加パラメータ（hpPct/pierce/dmgMul/defDown/atkMul/duration等）
}
```

発動時に `c._enragedShown` フラグで激昂演出（1回のみ）。  
デバフ系は `c._defDebuffFlat` / `c._healBlockedTurns` / `c._evadeSealedTurns` / `c._atkDebuffMul` / `c._atkDebuffTurns` で管理。

### 16-3. 転生スキル「深淵の境地」

`unlockRank:20` フィールドで worldRank20 未満の取得をサーバー・UI両面でガード。  
UI: worldRank20未満は非表示（専用ロックカードを表示）。  
worldRank20到達時: `_showRebirthMilestoneModal` で専用演出。

## 17. S48 追加仕様

### 17-1. 初回起動ファーストガイド

- **表示条件**: `gs.meta.isFirstPlay === true`（初回のみ）
- **非表示条件**: 既存セーブあり / スキップ / 「はじめる！」タップ
- **3ページ構成**: STEP1「まず潜ろう」→ STEP2「鑑定して売ろう」→ STEP3「稼いで強くなろう」
- **操作**: 次へ / 戻る / スキップ（STEP1のみ）/ はじめる！（STEP3のみ）

### 17-2. 帰還後「次の一手」ヒントUI

帰還成功リザルトの「閉じる」後に冒険タブ最上部へ1件表示。8秒でフェード。タップで対応タブへ遷移。

| 優先順位 | 条件 | 表示内容 | 遷移先 |
|---|---|---|---|
| 1 | 未鑑定品あり | 「未鑑定品を鑑定しよう（N個）」 | 格納庫タブ |
| 2 | 空き棚 + 出品可能装備あり | 「棚に商品を並べよう（空きN枠）」 | ショップタブ |
| 3 | CP不足 + 強化できる装備あり | 「装備を強化してCPを上げよう」 | 格納庫タブ |
| 4 | 傭兵代不足（傭兵同行中のみ） | 「放置収入を待とう」 | ショップタブ |
| 5 | デフォルト | 「N階層へ潜入しよう」 | 冒険タブ |

### 17-3. 売却快感システム

売却が発生した全箇所（通常棚・常連・裏取引）で演出を統一。

| 演出 | 条件 |
|---|---|
| goldPop（右上固定・18px） | 全売却 |
| topBar発光 | 全売却 |
| ゴールドカウントアップ | 1000G以上の売却 |
| 青バナー（2秒） | rare以上（鑑定済み） |
| 紫バナー（3秒） | epic以上（鑑定済み） |
| 橙バナー + 全画面フラッシュ | legendary（鑑定済み） |
| タブバッジ点滅 | rare以上・ショップタブ以外にいる時 |

ボス演出中・LvUP演出中はバナーを表示しない。

### 17-4. 実績解除トースト通知

`claimAchievement()` が呼ばれると自動表示。

- **位置**: 画面下部（bottom:142px）スライドアップ
- **表示時間**: 4500ms → 自動フェードアウト
- **最大同時表示**: 3件（キュー制・0.4秒間隔）
- **タップ**: 既存モーダルをクリアして実績モーダルを開く
- **対象外**: `retroCheckAchievements` による遡及付与（サイレント）

## S44（2026-03-23）

### バグ修正
- **generateEnemy: poisonRate/burnRate 欠落バグ修正**（重大）
  - `generateEnemy()` の return オブジェクトに `poisonRate`・`burnRate` が含まれておらず、全階層で敵からの毒付与・5階の熱傷付与が完全に機能していなかった
  - スプレッド構文 `...(base.poisonRate ? { poisonRate: base.poisonRate } : {})` で条件付きコピーする形に修正
  - 影響敵: スライム(20%)・ゴブリン(15%)・若き竜(30%)・毒蛙(55%)・疫病兵(35%)の毒付与、熔岩蜥蜴(50%)・地獄甲冑(30%)・業炎竜(60%)の熱傷付与

### バランス調整
- **inferno_drake burnRate**: 60 → 40（poisonRate/burnRateバグ修正により有効化されたため同時緩和）
- **rollRarity ボーナス上限**: `depth×0.5 + luk×0.3 + rarityBonus` に `Math.min(40, ...)` キャップ追加
  - 高luk深部でのLegendary率: 最大約24% → 約19% に抑制
  - luk特化ビルドの旨みは維持

### ログカラー移行（残存箇所を解消）
- 薬草師の帰還時HP回復ログ: `HP+N` → `healSpan` 適用
- 祠のHP回復ログ: `HP+N` → `healSpan` 適用

### INFERNO-SET実装（業炎のアクセセット・新テーマ・セット効果）

#### 新規アイテム3種追加
- `inferno_amulet`（業炎の護符 / neck / 1100G / STR+6 INT+4）
- `inferno_bangle`（業炎の腕輪 / bracelet / 950G / STR+4 VIT+5）
- `inferno_seal`（業炎の刻印 / seal / 1050G / STR+5 DEF+4 LUK+2）

#### ドロップテーブル更新
- 熔岩蜥蜴 drop に `inferno_bangle` 追加
- 地獄甲冑 drop に `inferno_seal` 追加
- 業炎竜 drop に `inferno_amulet` 追加
- 深淵の炎皇 drop に新3種追加
- generateDrop 5階プールに新3種追加

#### コレクションテーマ追加（22・23テーマ目）
- `inferno_jewelry`「業炎の装身具」: inferno_amulet / inferno_bangle / inferno_ring（neck/bracelet/ring）
- `inferno_ward`「業炎の封印」: ashcloak / inferno_seal（cloak/seal）
- コレクション総数: 2,028 → **2,132**
- 報酬設計: jewelry=STR/LUK/クリ特化、ward=STR/DEF/INT特化（ともにL帯でmaxHpPct+1）

#### セット効果追加（inferno_set「業炎の覇道」）
- 対象: inferno_ring / ashcloak / inferno_amulet / inferno_bangle / inferno_seal（全5種）
- 2部位: STR+6・熱傷ダメ-30%（`burnDmgReduce`）
- 3部位: 攻撃時15%で熱傷付与（`burnRateBonus`）
- 5部位: 熱傷中の敵への与ダメ+20%（`burnDmgBonus`）

#### 新フィールド実装
- `setBurnDmgReduce`: `applyBurnDamage` 内で熱傷ダメを軽減
- `setBurnRateBonus`: `doBattle` 内で攻撃時に敵へ熱傷付与（`en._burned`）
- `setBurnDmgBonus`: `doBattle` 内でダメージ計算時に熱傷中の敵への与ダメ加算
- 敵熱傷ダメージ処理追加: `en._burned` 時に毎ターン `maxHp×3%` スリップ（敵毒処理と対称）
